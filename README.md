# Kalkulator Diskon

**Kalkulator Diskon** adalah aplikasi berbasis GUI sederhana yang dibuat menggunakan **Python** dan **Tkinter**. Program ini dirancang untuk menghitung harga akhir setelah diskon berdasarkan harga asli dan persentase diskon yang diberikan.

---

## Fitur

- **Input Harga Asli**: Masukkan harga asli produk.
- **Input Persentase Diskon**: Masukkan persentase diskon (0-100%).
- **Hitung Harga Setelah Diskon**: Program akan menghitung harga akhir setelah diskon secara otomatis.
- **Tampilan GUI Interaktif**: Menggunakan library Tkinter untuk antarmuka pengguna yang sederhana dan mudah digunakan.
- **Validasi Input**: Program menangani input yang tidak valid dengan pesan error.

---

## Teknologi yang Digunakan

- **Python**: Bahasa pemrograman utama.
- **Tkinter**: Library bawaan Python untuk membuat antarmuka grafis (GUI).

---

## Instalasi dan Penggunaan

1. **Persyaratan**:
   - Pastikan Anda telah menginstal Python 3 di komputer Anda.
   - Tkinter biasanya sudah tersedia secara default dengan instalasi Python.

2. **Langkah Instalasi**:
   - Clone repository ini atau salin file `main.py`.
   - Pastikan file tersebut tersimpan di direktori lokal Anda.

3. **Menjalankan Program**:
   - Buka terminal atau IDE Python Anda.
   - Jalankan program dengan perintah:
     ```bash
     python main.py
     ```
   - Antarmuka aplikasi akan muncul, dan Anda dapat langsung menggunakan aplikasi.

---

## Cara Menggunakan

1. Masukkan harga asli produk pada field **Original Price (Rp)**.
2. Masukkan persentase diskon pada field **Discount (%)**.
3. Klik tombol **Calculate**.
4. Harga akhir setelah diskon akan ditampilkan di bawah tombol dalam format **Rp x,xxx.xx**.
5. Jika ada kesalahan input (contoh: nilai negatif, non-numerik, atau diskon lebih dari 100%), pesan error akan muncul.

---

## Contoh Penggunaan

### Input:
- **Original Price**: 100000  
- **Discount**: 20  

### Output:
- **Final Price**: Rp 80,000.00
